Report Pages Guide

Fighter Profile

This page provides an in-depth analysis of a selected fighter within a chosen weight class. It combines fighter background details, performance metrics, and fighting style characteristics. Data can be explored for a specific fight or aggregated across multiple fights.

  • Fighter Bio:
    • Nationality
    • Age
    • Weight
    • Height
    • Fight Record
  • Performance Metrics:
    • Attacking Accuracy:
      • Overall striking accuracy
      • By strike type:
        • Punches
        • Kicks
        • Knees
        • Elbows
      • Takedowns
    • Defense Rate:
      • Overall striking defense
      • By strike type:
        • Punches
        • Kicks
        • Knees
        • Elbows
      • Takedowns
  • Fighting Style:
    • Strike volume distribution (by strike type)
    • Activity level:
      • Strikes thrown per round
      • Takedowns attempted per round
    • Fight outcomes distribution:
      • By result
      • By round
      • By finish type

Roster Analysis I

This page provides an overall demographic analysis of the roster, highlighting key patterns and distributions.

  • Total number of fighters in the roster
  • Countries of origin:
    • Overall count of represented countries
    • Geographic heatmap of fighter origins
    • Table of countries ranked by number of fighters
  • Roster growth over time:
    • Line chart depicting cumulative growth of roster by year
  • Weight-class distribution of fighters
  • Age group distribution of fighters

Roster Analysis II

This page highlights fighter rankings based on selected criteria, such as fight finish type and weight class.

  • Top 10 – Longest Winning Streaks
  • Top 10 – Most Career Wins
  • Leaders in Striking Output (highest strikes landed per minute)

Fights Analysis I

This page provides an analysis of fight finishes from the perspective of events.

  • Total number of events
  • Total number of fights
  • Total number of title fights
  • Geographic spread of events (with drill-down capability):
    • Number of hosting countries
    • Number of hosting cities
    • Number of hosting venues
  • Finish types per event
  • Overall distribution of fight finish types
  • Breakdown of finish techniques:
    • Submissions
    • Knockouts

Fights Analysis II

This page analyzes fight dynamics across weight classes.

  • Total number of fights
  • Total number of strikes thrown

  • Total number of attempted submissions
  • Bar chart depicting average attempted takedowns per round
  • Column chart depicting total and average number of strikes
  • Column chart depicting fight finish types per round
  • Column chart depicting striking and takedown accuracy
