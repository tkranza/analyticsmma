Report Pages Guide

Page 1 - Fighter Profile

For the chosen fighter inside a chosen weight-class it shows fighter bio, performance metrics, and a fighting style. Can be analyzed for a specific fight or across selected range of multiple fights.

  • Fighter bio:
    • Nationality
    • Age
    • Weight
    • Height
    • Fight Record
  • Performance metrics:
    • Attacking accuracy:
      • Overall striking
      • Type of strike:
        • Punches
        • Kicks
        • Knees
        • Elbows
      • Takedown
    • Defense rate:
      • Overall striking
      • Type of strike:
        • Punches
        • Kicks
        • Knees
        • Elbows
      • Takedown
  • Fighting style:
    • Volume distribution of strikes thrown
    • Activity:
      • Strikes thrown per round
      • Takedowns thrown per round
    • Fights outcome distribution per:
      • Result
      • Round
      • Finish

Page 2 - Roster Analysis I

Shows overall roster analysis.

  • Number of fighters
  • Countries of origin of fighters:
    • Overall number
    • Heatmap
  • Table of countries ordered by number of fighters
  • Line chart depicting cumulative growth of roster by year
  • Distribution of fighters across weight-class
  • Distribution of fighters across age groups

Page 3 - Roster Analysis II

Shows rankings based on selected criteria - Fight finish type & Weight-class.

  • TOP 10 - Longest winning streak
  • TOP 10 - Most wins
  • Most strikes landed per minute

Page 4 - Fights Analysis I

Fight finishes analysis from perspective of events.

  • Number of events
  • Number of fights
  • Number of event-hosted countries
  • Number of event-hosted cities
  • Types of finishes per event
  • Distribution of fight finish types
  • Distribution of specific finish techniques:
    • Submissions
    • Knockouts

Page 5 - Fights Analysis II

Fight dynamics analysis from perspective of weight-classes.

  • Number of fights
  • Number of strikes thrown
  • Number of attempted takedowns
  • Number of attempted submissions
  • Bar chart depicting average attempted takedowns per round
  • Bar chart depicting total and average number of strikes
  • Bar chart depicting fight finish types per round
  • Bar chart depicting thrown and landed strikes per fight finish type
  • Total fights per weight-class
